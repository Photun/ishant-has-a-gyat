# Product Positioning

## Working Product Name
Launchpad Skills

## One-Line Description
A modern career acceleration platform that helps learners build practical, job-ready digital skills through structured courses, guided projects, and mentor-backed learning.

## Brand Direction
The product should feel:
- modern
- credible
- direct
- outcome-focused
- premium without sounding corporate or robotic

## Core Messaging Pillars
1. Learn practical skills that map to real careers
2. Build a portfolio through guided, applied work
3. Move faster with structured content and expert support
4. Gain confidence through clarity, repetition, and execution

## Homepage Messaging
### Hero headline
Build job-ready digital skills with clarity.

### Hero subheadline
Structured courses, guided projects, and practical learning designed to help ambitious learners move from curiosity to capability.

### Primary CTA

## Featured Program Copy
### UX/UI Design

### Learn by doing
Every program emphasizes applied work, not passive consumption.

### Flexible structure
Learn on a schedule that works without sacrificing momentum.

### Career-focused outcomes
Build confidence, fluency, and a body of work you can actually present.
