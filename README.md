# ishant-has-a-gyat

A polished static website project built with exported HTML, CSS, JavaScript, and media assets.

## Overview
This repository contains a multi-page educational landing site with:
- a responsive homepage and navigation
- course and webinar pages
- branded visual assets and SVG illustrations
- contact and newsletter forms
- media-rich showcase sections

## Tech Stack
- HTML
- CSS
- JavaScript
- SVG / PNG / JPG / MP4 assets

## Project Structure

```text
.

└── favicon_d1_res.ico
```

## Local Development
Because this is a static site export, you can preview it locally with any simple static server.

### Option 1: Python
```bash
python -m http.server 8000
```
Then open `http://localhost:8000`.

### Option 2: Node.js
```bash
npx serve .
```
